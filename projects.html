Yelp Clone

Yelp Clone is a personal project I created using the PERN stack (Postgres, Express, Node, and React) with the goal of building a simplified version of Yelp. The platform allows users to perform various actions such as writing reviews on restaurants, creating new restaurants, and editing any information already on the platform. To accomplish this, I created a database using Postgres to store information inputted by the user, and utilized Express, Node, and React for both backend logic and front-end user interface. My aim was to create a user-friendly platform with an intuitive design, that emulated the functionalities of Yelp, while also providing a unique and personalized user experience.
